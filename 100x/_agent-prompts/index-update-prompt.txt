첨부된 파일들을 사용하여 100x Daily Wrap 데이터를 업데이트해줘.

지시사항: 100x-index-agent.md 파일의 모든 규칙을 엄격히 따라줘.

수행할 작업:
1. '신규 리포트 HTML' 파일에서 정보를 추출해 '신규 메타데이터 JSON' 파일을 생성해줘.
2. '기존 인덱스 JSON' 파일 맨 앞에 새로운 메타데이터 파일명을 추가해서 '업데이트된 인덱스 JSON' 파일을 만들어줘.

결과물:
아래 두 개의 JSON 파일에 대한 전체 코드를 각각 제공해줘.
1. 신규 메타데이터 JSON 파일
2. 업데이트된 인덱스 JSON 파일


### **앞으로 제공해야 할 파일**

위 프롬프트와 함께 저에게 작업을 요청하실 때는, 아래 **두 가지 파일**만 제공해주시면 됩니다.

1.  **신규 리포트 HTML:** 새로 추가할 리포트의 원본 HTML 파일 (예: `2025-07-17_100x-daily-wrap.html`)
2.  **기존 인덱스 JSON:** 현재까지의 리포트 목록이 담긴 `reports-index.json` 파일
